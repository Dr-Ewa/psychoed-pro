## One-time setup
Open Terminal, then run these two commands:

    cd ~/Desktop/psychoed-app
    npm install

## Run the app (every time)
Open Terminal, then run:

    cd ~/Desktop/psychoed-app
    npm run dev

Then open your browser to: http://localhost:5173

## Stop the app
Press Ctrl+C in the Terminal.

## API Key
Enter your OpenAI API key in the app's settings area (the key icon).
Get a key at: https://platform.openai.com/api-keys
The key is saved in your browser and remembered between sessions.
